Uso Correto: ./update.sh <OPERATION_LABEL>
Operation Labels:
	-help: Imprime ajuda/uso correto do script
	-build-leggor: Atualiza e faz build do container leggoR
	-pull-docker: Atualiza container leggoR
	-update-pautas: Baixa dados atualizados de pautas
	-update-data: Baixa dados atualizados para o leggoR (versão nova)
	-process-data: Process dados do leggoR
	-fetch-data: Baixa dados para o leggoR (versão antiga)
	-fetch-autores: Baixa dados dos autores das proposições
	-fetch-props: Baixa dados de proposições
	-fetch-emendas: Baixa dados de emendas
	-fetch-comissoes: Baixa dados de comissões
	-update-emendas: Atualiza dados de emendas com distâncias atualizadas
	-update-db-dev: Importa dados atualizados para o Banco de Dados do Backend Dev
	-update-db-prod: Importa dados atualizados para o Banco de Dados do Backend Prod
	-run-basic-pipeline: Roda pipeline básico de atualização de dados do Leggo (sem análise de emendas)
	-run-full-pipeline: Roda pipeline completo de atualização de dados do Leggo
	-run-pipeline-leggo-content: Roda pipeline para análise das Emendas
	-build-leggo-trends: Atualiza e faz o build do Container Leggo Trends
	-fetch-leggo-trends: Computa dados para a Pressão usando o Leggo Trends
	-build-versoes-props: Atualiza e faz o build do Container Versões Props
	-fetch-versoes-props: Computa dados para a Pressão usando o Versões Props
	-build-leggo-content: Atualiza e faz o build do Container Leggo Content
	-process-leggo-content: Processa dados de textos/conteúdo usando o Leggo Content
	-process-anotacoes: Processa dados de anotações
	-update-db-insights-dev: Importa dados atualizados de Insights para o Banco de Dados do Backend Dev
	-update-db-insights-prod: Importa dados atualizados de Insights para o Banco de Dados do Backend Prod
	-atualiza-parlamentares: Atualiza os dados dos parlamentares
	-process-entidades: Processa dados de entidades
	-generate-backup: Gera pasta com backup dos csvs
